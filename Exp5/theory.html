Shift Reduce Parsing

Shift Reduce Parsing is a bottom-up parsing technique used in compiler design. It constructs the parse tree from leaves to root by repeatedly performing shift and reduce operations on an input string.

Operations in Shift Reduce Parsing

  • Shift: Move the next input symbol from the input buffer to the stack.
  • Reduce: Replace a sequence of symbols (handle) on the stack with a non-terminal according to a grammar rule.
  • Accept: The input string is successfully parsed.
  • Error: No valid shift or reduce action exists.

Working Principle

The parser scans the input from left to right. Symbols are shifted onto the stack until a handle is found. The handle is then reduced using grammar productions. This process continues until the start symbol remains on the stack and the input is exhausted.
